Numerical parameters are extracted with a combination of LLM prompts (defined in file .../Modules/LLM.py) and REGEX patterns (defined in file .../Modules/functions_PARAMETERS.py). Check available numerical parameters bellow (parameter -> example):

Categorical parameters are extracted with function regex.search(input_string, cat_term), where argument "input_string" represents text documents from the corpus being scanned (sentences or paragraphs), and "cat_term" represents catergorical terms of interest that are defined in a JSON file in .../Inputs/ner_rules.json.
